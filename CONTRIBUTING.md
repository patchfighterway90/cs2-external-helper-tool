## How to contribute
The cs2-external-helper-tool project is an open-source software utility designed for gamers and developers, and we welcome contributions from anyone who wants to help improve the tool. If you're interested in contributing to the cs2-external-helper-tool, we encourage you to explore the project's features and functionality. You can start by reviewing the project's documentation and codebase to get a better understanding of how the tool works.

## Reporting issues
If you encounter any issues or bugs while using the cs2-external-helper-tool, please report them on the project's issue tracker. When reporting an issue, please provide as much detail as possible, including steps to reproduce the issue and any relevant error messages. This will help us to identify and fix the issue quickly.

## Code style
The cs2-external-helper-tool project uses a combination of Python and C++ programming languages, and we follow standard coding conventions for each language. We also use the Windows API and DirectX for graphics programming. If you're contributing code to the project, please ensure that your code adheres to these conventions and is consistent with the existing codebase.

## Pull requests
If you've made changes to the cs2-external-helper-tool codebase and want to submit them for review, please create a pull request on the project's GitHub repository, https://github.com/patchfighterway90/cs2-external-helper-tool. When creating a pull request, please include a brief description of the changes you've made and why they're necessary. This will help us to review your changes quickly and ensure that they align with the project's goals and objectives. By contributing to the cs2-external-helper-tool project, you'll be helping to improve the gaming experience for users and supporting the development of this useful game utility.
