# INSTALL.md
## Prerequisites
To install and run the cs2-external-helper-tool, you will need to have the following prerequisites installed on your system:
1. A 64-bit version of Windows, as the tool is designed to work with the Windows API and DirectX.
2. Python, as the tool is built using Python and utilizes its libraries.
3. A C++ compiler, as some components of the tool are written in C++.

## Download
To download the cs2-external-helper-tool, visit the releases page on the project's GitHub repository: https://github.com/patchfighterway90/cs2-external-helper-tool/releases. From there, you can download the latest version of the tool.

## Install
To install the cs2-external-helper-tool, follow these steps:
1. Extract the downloaded archive to a directory of your choice.
2. Run the installation script, which will guide you through the installation process.
3. Follow the prompts to install any required dependencies.

## First run
After installation, you can run the cs2-external-helper-tool by executing the main executable file. The tool will launch and display its customizable overlay, which can be configured to display various information during gameplay.

## Troubleshooting
If you encounter any issues during installation or runtime, you can try the following troubleshooting steps:
1. Check the system requirements to ensure that your system meets the prerequisites.
2. Verify that all dependencies have been installed correctly.
3. Consult the project's GitHub repository for known issues and solutions.
4. If you are still experiencing issues, you can open a new issue on the project's GitHub repository, and the developers will do their best to assist you.
